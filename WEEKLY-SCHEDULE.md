Week 1 — Fundamentals and Lab Setup
Learning: Course 1 modules 1–3; Computer Networking A Top‑Down Approach chapters on application and transport layers.

Lab: Install GNS3 or Packet Tracer; create a 3‑node topology (2 hosts + 1 switch/router).

Deliverable: projects/enterprise-topology/week-1/ — topology file, topology diagram (PNG), and a 1‑page runbook.

Time estimate: 8–12 hours

Week 2 — IPv4 Addressing and VLANs
Learning: Course 1 modules on IPv4, subnetting, and VLAN concepts; TCP/IP Illustrated Vol 1 chapter on IP.

Lab: Configure VLANs and inter‑VLAN routing in GNS3; capture an inter‑VLAN ping with Wireshark.

Deliverable: projects/enterprise-topology/week-2/ — configs, pcap, and a 1‑paragraph packet flow explanation.

Time estimate: 8–12 hours

Week 3 — Basic Routing and Troubleshooting
Learning: Course 1 routing modules (static routing, OSPF intro); Network Warrior troubleshooting chapters.

Lab: Configure static routes and OSPF on the topology; simulate a link failure and document troubleshooting steps.

Deliverable: projects/enterprise-topology/week-3/ — configs, troubleshooting log, and remediation steps.

Time estimate: 8–12 hours

Week 4 — Review and Practice Exam
Learning: Course 4 practice quizzes and weak area review.

Lab: Timed troubleshooting challenge; re‑run Week 2 and Week 3 labs under a 60‑minute timer.

Deliverable: projects/enterprise-topology/week-4/ — 5‑minute demo video or GIF and a short reflection note.

Time estimate: 6–10 hours

Week 5 — Python for Networking
Learning: Course 2 Python for network engineers modules; basic Netmiko examples.

Lab: Write a Python script to SSH to a virtual router and collect interface status.

Deliverable: projects/automation-dashboard/week-5/ — scripts/get_interfaces.py, README, sample JSON output.

Time estimate: 8–12 hours

Week 6 — Data Normalization and Flask Prototype
Learning: Course 2 NAPALM and JSON parsing modules; targeted TCP/IP Illustrated sections as needed.

Lab: Normalize device outputs into JSON and store in a small SQLite DB; expose a REST endpoint with Flask.

Deliverable: projects/automation-dashboard/week-6/ — Flask app skeleton, sample API response, README.

Time estimate: 8–12 hours

Week 7 — Ansible for Network Devices
Learning: Course 2 Ansible modules for network automation.

Lab: Create Ansible playbooks to provision VLANs and push a simple config change to virtual devices.

Deliverable: ansible/week-7/ — playbooks, inventory, runbook showing idempotency tests.

Time estimate: 8–12 hours

Week 8 — CI Integration and Tests
Learning: Course 4 CI and automation best practices; linting and basic unit tests for scripts.

Lab: Add GitHub Actions to run a linter and a smoke test on push (simulate device responses).

Deliverable: .github/workflows/ci.yml, test script, and documentation of CI behavior.

Time estimate: 6–10 hours

Week 9 — Advanced Routing or Cloud Networking
Learning: Course 3 modules on BGP or cloud networking depending on your target role.

Lab: Implement BGP peering in GNS3 or deploy a VPC network in a cloud free tier and configure subnets and route tables.

Deliverable: projects/enterprise-topology/week-9/ — BGP configs or cloud network diagram and config snippets.

Time estimate: 8–12 hours

Week 10 — Security Basics and ACLs
Learning: Course 4 security modules and Network Warrior chapters on access control and VPN concepts.

Lab: Implement ACLs to restrict traffic; capture and document allowed versus blocked flows.

Deliverable: projects/enterprise-topology/week-10/ — ACL configs, pcap examples, and a security checklist.

Time estimate: 8–12 hours

Week 11 — Portfolio Polish and Mock Interviews
Learning: Revisit weak topics; prepare STAR stories for troubleshooting incidents.

Lab: Finalize both portfolio projects; create demo GIFs and short walkthrough videos.

Deliverable: projects/ — polished repos for Automation Dashboard and Enterprise Topology, demo GIFs, and weekly-updates.md entries.

Time estimate: 8–12 hours

Week 12 — Certification Practice and Job Applications
Learning: Take practice exam for Network+ or CCNA; review exam objectives.

Lab: Reproduce one end‑to‑end scenario from your portfolio and record a 5‑minute demo.

Deliverable: Updated resume, five tailored job applications, and a final repo README with links to demos.

Time estimate: 6–12 hours
